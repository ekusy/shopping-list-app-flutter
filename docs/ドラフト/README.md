# ドラフト

このフォルダ配下のドキュメントは **検討中のドラフト** であり、現状の実装を反映した
`docs/内部設計/`・`docs/外部仕様/` とは区別する。

- 実装に着手し設計が確定した時点で、該当ドキュメントを正式な場所
  （`docs/内部設計/` 等）へ移動・統合する。
- ドラフト段階では既存の正式ドキュメント（アーキテクチャ概要・ドメインモデル等）を
  変更しない。

| サブフォルダ | 内容 | 状態 |
|---|---|---|
| `AI提案機能/` | Gemini を活用した購入忘れ指摘・次期購入提案システムの設計検討 | 検討中 |
| `機能構想/` | 今後実装したい機能の構想（よく買う物リスト / 購入履歴確認 / 画像の Storage 移行 / 通知 / マネタイズ） | 検討中 |
