MCP サーバー
Dart MCP Server Tier 0 — 基盤

dart analyze・テスト実行・pub.dev 検索・コード整形・シンボル解決。Flutter 開発の中核ツール。

静的解析自動テストデバッグドキュメント
APIキー不要
Node.js 不要
Dart 3.9+
インストール
claude mcp add --scope project --transport stdio dart -- dart mcp-server
配布元: flutter.dev / Google
Context7 Tier 1 — 汎用

pub.dev・Flutter・任意ライブラリのドキュメントをリアルタイム参照。バージョン差異・新 API 対応に強い。

ドキュメント仕様作成アーキテクチャ
APIキー不要
Node.js 不要
インストール
claude mcp add --transport http context7 https://mcp.context7.com/mcp
配布元: Upstash / context7.com
Sequential Thinking Tier 1 — 汎用

複雑な設計・デバッグ・要件分解を段階的推論で解決。アーキテクチャ検討や難易度の高いバグ調査に有効。

アーキテクチャ要件定義デバッグ
APIキー不要
Node.js 必要
インストール
claude mcp add sequential-thinking npx -- -y @modelcontextprotocol/server-sequential-thinking
配布元: Anthropic / MCP公式
Firebase MCP Tier 2 — 構成次第

Firestore・Auth・Crashlytics・FCM を AI から直接操作。Firebase バックエンドを使う Flutter アプリに最適。

デバッグCI/CD自動テスト
APIキー不要（firebase login）
Node.js 必要
インストール
claude mcp add firebase npx -- -y firebase-tools@latest mcp
配布元: Google / Firebase公式

Notion MCP Tier 3 — チーム連携

要件定義仕様作成ドキュメント

Linear MCP Tier 3 — チーム連携

チケット・スプリント・ロードマップ操作。Figma デザインリンクとの連携も可。要件→チケット変換ワークフローに有効。

要件定義タスク管理CI/CD
Linear APIキー必要
Node.js 必要
インストール（LINEAR_API_KEY が必要）
claude mcp add linear npx -- -y linear-mcp-server
配布元: Linear / 公式
Sentry MCP Tier 3 — チーム連携

クラッシュレポート・スタックトレースを AI から直接参照してデバッグ。本番エラーの原因特定を高速化。

デバッグ静的解析CI/CD
Sentry Auth Token 必要
Node.js 不要（HTTP）
インストール（OAuth で自動認証）
claude mcp add --transport http sentry https://mcp.sentry.dev/mcp
配布元: Sentry / 公式
Slack MCP Tier 3 — チーム連携

チャンネル検索・DM・メッセージ投稿。議事録→仕様への変換や、Slack スレッドから要件を抽出するワークフローに活用。

要件定義ドキュメント
